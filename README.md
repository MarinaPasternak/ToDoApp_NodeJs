#  ToDo App
This is a web application for managing tasks. 
The user can add or delete tasks.

## Technologies Used
- Node.js
- Express.js
- jQuery
- Axios

### Install the dependencies.
```
npm install
```

### Start the development server.
```
npm run serve
```
